## 🚀 功能扩展建议

### P0 - 核心功能（强烈建议）

#### 1. 评论系统 💬

**价值**: 提升用户互动，增加网站活跃度

**实现方案**:
```prisma
model Comment {
  id        String   @id @default(uuid())
  content   String
  postId    Int
  post      Post     @relation(fields: [postId], references: [id], onDelete: Cascade)
  authorId  String
  author    User     @relation(fields: [authorId], references: [id])
  parentId  String?  // 支持回复
  parent    Comment? @relation("CommentReplies", fields: [parentId], references: [id])
  replies   Comment[] @relation("CommentReplies")
  status    String   @default("PENDING") // PENDING, APPROVED, REJECTED
  createdAt DateTime @default(now())
}
```

**API 端点**:
- `GET /api/posts/:id/comments` - 获取文章评论
- `POST /api/posts/:id/comments` - 发表评论
- `PATCH /api/comments/:id` - 更新评论（作者）
- `DELETE /api/comments/:id` - 删除评论
- `POST /api/comments/:id/approve` - 审核评论（管理员）

**预计工作量**: 2-3 天

#### 3. RSS/Atom Feed 📡

**价值**: 支持 RSS 阅读器订阅，老用户召回

**实现方案**:
```typescript
// GET /feed.xml
import { Feed } from 'feed';

const feed = new Feed({
  title: '博客名称',
  description: '博客描述',
  id: 'https://yourblog.com/',
  link: 'https://yourblog.com/',
  language: 'zh-CN',
  favicon: 'https://yourblog.com/favicon.ico',
  copyright: 'All rights reserved 2026',
  updated: new Date(),
});

// 添加最新文章
posts.forEach(post => {
  feed.addItem({
    title: post.title,
    id: post.slug,
    link: `https://yourblog.com/post/${post.slug}`,
    description: post.excerpt,
    content: post.content,
    author: [{ name: post.author.name }],
    date: post.publishedAt,
  });
});
```

#### 6. 搜索功能 🔎

**价值**: 帮助用户快速找到内容

**实现方案**:

| 方案 | 说明 | 成本 |
|------|------|------|
| PostgreSQL 全文搜索 | 内置 tsvector/tsquery | 低 |
| Meilisearch | 开源搜索引擎 | 中 |
| Algolia | SaaS 搜索服务 | 高 |

**PostgreSQL 实现**:
```prisma
model Post {
  // ...
  searchVector String? @db.Text // tsvector
  
  @@index([searchVector])
}
```

```typescript
// 搜索查询
const results = await prisma.$queryRaw`
  SELECT * FROM posts
  WHERE search_vector @@ to_tsquery('chinese', ${query})
  ORDER BY ts_rank(search_vector, to_tsquery('chinese', ${query})) DESC
`;
```

**预计工作量**: 1-2 天

#### 8. 图片优化 🖼️

**价值**: 提升加载速度，节省带宽

**实现方案**:
```typescript
import sharp from 'sharp';

// 上传时生成多尺寸
async function processImage(buffer: Buffer) {
  const sizes = [
    { name: 'thumbnail', width: 300 },
    { name: 'medium', width: 768 },
    { name: 'large', width: 1920 },
  ];
  
  const results = {};
  for (const size of sizes) {
    const optimized = await sharp(buffer)
      .resize(size.width, null, { withoutEnlargement: true })
      .webp({ quality: 80 })
      .toBuffer();
    results[size.name] = optimized;
  }
  return results;
}
```

**前端响应式图片**:
```html
<picture>
  <source 
    srcset="/img/thumb.webp, /img/medium.webp 2x"
    type="image/webp"
  />
  <img src="/img/thumb.jpg" alt="..." loading="lazy" />
</picture>
```

**预计工作量**: 1-2 天
